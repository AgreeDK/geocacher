## [0.2.0] — Beta — 2026

### Tilføjet
- **Avanceret filter dialog** med 3 faner (Generelt, Datoer, Attributter)
- **Filter toolbar** — 🔍 Filter (Ctrl+F) og ❌ Slet filter
- **ROT13 hint dekodning**
- **Søg i logs** med fremhævning
- **Status ikoner** — ✅ fundet, ❌ DNF, 🔒 arkiveret, ⚠️ utilgængelig
- **Klik på GC kode** → geocaching.com
- **Klik på koordinat** → kortapp
- **Højreklik kontekstmenu** i cachelisten
- **Valgfri kortapp** — Google Maps eller OpenStreetMap
- **Opdater fund fra reference database** (My Finds PQ)
- **Favorit ★ kolonne**
- **Waypoint CRUD** — tilføj, rediger, slet caches manuelt
- **Kolonnevælger** — 17+ kolonner

## [0.1.0] — Beta — 2026

### Tilføjet
- Import af GPX og Pocket Query ZIP filer
- Interaktivt OSM kort med farvekodet pins
- Filter engine med 18 filtertyper og AND/OR logik
- Gemte filterprofiler
- Flere databaser med manager dialog
- Centerpunkt per database
- Cache detaljer med beskrivelse, hints og logs
- Indstillinger — hjemkoordinat, afstandsenhed, kortapp
